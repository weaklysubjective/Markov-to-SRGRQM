# Special Relativity from CA/MM (Poset-Based)

This repo shows **SR kinematics** emerging from *Conscious Agents / Markov Matrix* updates using only:
- one-step-per-tick causal updates,
- a **partial order** (no backward-in-time edges),
- and **counts** of causal relations (no metric assumed).

Core scripts live in `src/`. A full per-test guide (CLI, math, intuition) is in **`src/README_SR.md`**.
